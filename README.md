# רשימת קניות משותפת — PWA

אפליקציית רשימת קניות מסונכרנת לאלעד ונויה, שנפתחת בטלפון כמו אפליקציה אמיתית
(מסך מלא, אייקון משלה, בלי ממשק דפדפן).

## מבנה הפרויקט

```
shopping-list-pwa/
├── server.js                  # Express + MongoDB — שרת וה-API לסנכרון
├── package.json
└── public/
    ├── index.html             # האפליקציה עצמה (כולל תגי ה-PWA)
    ├── manifest.webmanifest   # מה שהופך את האתר ל"אפליקציה"
    ├── sw.js                  # Service Worker — אופליין + תנאי התקנה
    └── icons/                 # אייקונים 192 ו-512
```

## הרצה מקומית (פיתוח)

```bash
# MongoDB מקומי ב-Docker, כמו שאתה כבר עובד:
docker run -d --name shopping-mongo -p 27017:27017 mongo:7

npm install
npm start
# נפתח ב- http://localhost:3000
```

## פרודקשן

1. צור Cluster חינמי ב-MongoDB Atlas וקח את ה-connection string.
2. פרוס את הפרויקט לכל פלטפורמה שנותנת HTTPS — Render / Railway / Fly.io
   (ל-PWA **חייבים** HTTPS; localhost פטור רק לפיתוח).
3. הגדר משתני סביבה:

```bash
MONGODB_URI="mongodb+srv://user:pass@cluster.xxx.mongodb.net"
LIST_TOKEN="סוד-משפחתי-כלשהו"   # אופציונלי אך מומלץ — ראה הערה למטה
```

> **הערת אבטחה:** בלי LIST_TOKEN, כל מי שמגלה את הכתובת יכול לקרוא ולערוך
> את הרשימה. אם תגדיר טוקן, תצטרך להוסיף ב-index.html את הכותרת
> `"x-list-token"` לשתי קריאות ה-fetch. לרשימת קניות זה לא קריטי,
> אבל זה תרגול טוב.

## התקנה בטלפונים (פעם אחת לכל טלפון)

**אנדרואיד (Chrome):** גלשו לכתובת → Chrome יציע "התקנת אפליקציה",
או דרך תפריט ⋮ → "הוספה למסך הבית" → "התקנה".

**אייפון (Safari בלבד!):** גלשו לכתובת → כפתור השיתוף (ריבוע עם חץ) →
"הוסף למסך הבית". בזכות התג `apple-mobile-web-app-capable` היא תיפתח
במסך מלא בלי ממשק ספארי.

מרגע ההתקנה — אייקון במסך הבית, פתיחה כאפליקציה לכל דבר.

## איך ה-PWA עובד (בקצרה)

- **manifest.webmanifest** עם `"display": "standalone"` — אומר למערכת ההפעלה
  לפתוח בלי כרום של דפדפן, ומגדיר שם/אייקון/צבעים.
- **sw.js** — שומר את הקבצים הסטטיים ב-cache (האפליקציה נפתחת גם בלי רשת),
  ובאנדרואיד הוא תנאי לכך ש-Chrome יציע התקנה. קריאות ה-API עוברות תמיד
  לרשת כדי שהסנכרון יישאר טרי.
- **תגי apple-*** ב-index.html — האייפון לא קורא manifest באופן מלא,
  אז הוא צריך את התגים הייעודיים שלו.
- **סנכרון:** המכשירים שואלים את השרת כל 4 שניות, ובכל חזרה לאפליקציה
  (visibilitychange) נטענת הגרסה העדכנית.
